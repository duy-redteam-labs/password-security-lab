# Password Security Lab (Hashing & Policy)

## Overview
Compare password storage approaches and demonstrate secure vs insecure implementations:
- MD5 vs BCrypt vs Argon2
- Benchmark dictionary/bruteforce on self-generated hashes (lab-only)
- Recommend password policy & hardening

## Scope (Lab-only)
Only use self-generated datasets. No real credentials.

## Tech Stack
- Python (argon2-cffi/bcrypt) or equivalent
- Report notebook/markdown
- Demo app (CLI/web)

## Deliverables
- Benchmark report: docs/reports/
- Demo app: scripts/ (or src/)
- Conclusions + policy recommendations
